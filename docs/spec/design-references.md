## 1. lemanapro.ru (Лемана Про, бывш. Leroy Merlin)

**Тип:** Каталог / E-commerce (DIY-ритейлер)

**Что ценного для CollegeLMS:**

- **Иерархическая структура каталога** — категории → подкатегории → товары. Аналог: Специальности → Курсы → Уроки.
- **Боковая панель фильтров** — чекбоксы, диапазон цен, бренды. Аналог: фильтрация расписания по группе/преподавателю/аудитории.
- **Карточки товаров** — фото, название, цена, рейтинг, кнопка "В корзину". Аналог: карточки курсов с прогрессом, статусом, кнопкой перехода.
- **Поиск с автоподсказками** — поиск по всему каталогу. Аналог: поиск по курсам, студентам, группам.

**Паттерны:**
- `категории → подкатегории → элемент` — навигация по учебной программе
- `фильтры + поиск + список` — страница расписания / журнала
- `карточка с CTA` — карточка курса на дашборде

## 2. vsk.ru (BCK — Военно-страховая компания)

**Тип:** Корпоративный сайт / Страхование

**Что ценного для CollegeLMS:**

- **Строгий официальный стиль** — консервативная типографика, умеренные цвета. Аналог: раздел документов, ведомости, отчёты.
- **Плитки продуктов с иконками** — "ОСАГО", "КАСКО", "Ипотека" — каждая как карточка с иконкой. Аналог: разделы LMS (Расписание, Оценки, Журнал, Материалы).
- **Личный кабинет с левым меню** — типичная структура: левая навигация + контент справа. Аналог: дашборд студента/преподавателя.
- **Калькуляторы** — расчёт стоимости полиса. Аналог: калькулятор среднего балла / посещаемости.

**Паттерны:**
- `левое меню + контент` — навигация для залогиненного пользователя
- `иконка + текст` — модули LMS на главной
- `официальная типографика` — ведомости, справки, печатные формы

## 3. vtb.ru (BТБ)

**Тип:** Банковский сайт

**Что ценного для CollegeLMS:**

- **Фирменный синий цвет** — солидный, вызывающий доверие. Перекликается с нашим College Blue (`#568cd6`). Пример, как синий работает в крупном бренде.
- **Крупные цифры-статистика** — "2 000+ вакансий", "2 место". Аналог: статистика на дашборде — "150 студентов", "12 групп", "8 преподавателей".
- **Карточки продуктов** — "дебетовая карта", "вклады", "кредиты" — каждая с заголовком, кратким описанием, кнопкой. Аналог: карточки курсов на главной.
- **Курсы валют как виджет** — компактная таблица с курсами. Аналог: виджет "ближайшие пары" / "дедлайны сдачи работ".
- **Акцент на доверии** — "Генеральная лицензия Банка России №1000". Аналог: "Лицензия Министерства образования".

**Паттерны:**
- `брендовый синий + белый` — цветовая схема
- `цифры-статистика` — дашборд
- `виджет с данными` — погода / пары / дедлайны
- `доверительные элементы` — лицензии, сертификаты

## 4. stgau.ru (Ставропольский государственный аграрный университет)

**Тип:** Сайт вуза

**Что ценного для CollegeLMS:**

- **Самый близкий аналог по типу** — учебное заведение. Прямой референс для структуры и контента.
- **Героический баннер** — крупное изображение университета/кампуса, название, слоган, кнопка CTA. Аналог: главная страница с логотипом колледжа.
- **Новостная лента** — вертикальные карточки: дата, заголовок, краткое описание, изображение, "Подробнее". Аналог: новости колледжа.
- **Раздел "Студенту"** — типовой портал: расписание, успеваемость, портфолио, заявления. Прямое совпадение с CollegeLMS.
- **Футер** — официальная информация: лицензия, контакты, реквизиты.

**Паттерны:**
- `героический баннер` — главная страница
- `новостная лента` — раздел новостей
- `раздел студента` — всё приложение CollegeLMS
- `официальный футер` — лицензия, контакты, реквизиты

## 5. mtsbank.ru (МТС Банк)

**Тип:** Банковский сайт

**Что ценного для CollegeLMS:**

- **Бирюзовый акцент** — фирменный цвет МТС. Наш аналог: Leaf Green (`#2d872d`). Пример, как один яркий акцент работает в дизайне.
- **Персонализация** — "Мы подготовили для вас персональные предложения". Аналог: дашборд студента с его группой, курсами, долгами.
- **Калькулятор кредита** — интерактивный: сумма, срок → ежемесячный платёж. Аналог: калькулятор баллов / среднего балла / рейтинга.
- **Продвижение приложения** — банк активно продвигает mobile app. Аналог: если будет мобильное приложение LMS.
- **Курсы валют** — таблица с покупкой/продажей. Аналог: таблица с оценками / баллами.

**Паттерны:**
- `цветовой акцент как бренд` — зелёный на белом
- `персонализированный блок` — "для вас"
- `калькулятор / интерактив` — расчёт баллов
- `promo app` — продвижение мобильной версии

## 6. tbank.ru (Т-Банк, бывш. Тинькофф)

**Тип:** Банковский сайт

**Что ценного для CollegeLMS:**

- **Жёлтый — ДНК бренда.** Самый узнаваемый фирменный цвет среди всех референсов. Пример, как один цвет становится идентичностью.
- **Tone of voice** — разговорный, человечный, без бюрократии. Аналог: сообщения об ошибках, пустые состояния, приветствия, подсказки.
- **Микро-анимация** — иконки "дышат", карточки реагируют на hover, переходы плавные. Аналог: анимация интерфейса LMS.
- **Минималистичный хедер** — только нужное, сразу к делу. Аналог: навигация на дашборде.
- **Главная = навигация** — никакой воды, сразу продукты. Аналог: главная страница LMS после логина.
- **Мобильный first** — все элементы работают на телефоне.

**Паттерны:**
- `один яркий цвет = ДНК` — фирменный стиль
- `разговорный tone of voice` — ошибки, пустые состояния
- `микро-анимация` — hover, загрузка, переходы
- `minimal хедер` — навигация

## 7. netology.ru (Нетология)

**Тип:** Образовательная платформа (онлайн-курсы)

**Что ценного для CollegeLMS:**

- **Самый релевантный референс.** Образовательная платформа — ближайший аналог CollegeLMS.
- **Карточки курсов** — чистая сетка: иконка направления, название, количество курсов. Аналог: каталог специальностей / курсов колледжа.
- **Социальное доказательство** — "4,88 из 5", "1,5 млн студентов", "4 000 партнёров". Аналог: "98% выпускников трудоустроены", "500+ студентов".
- **Категории курсов** — программирование, аналитика, дизайн, бизнес... Аналог: кафедры / специальности колледжа.
- **Платформа обучения** — планировщик, календарь, игры, тренажёры. Прямое совпадение с функциями LMS.
- **Фильтрация и поиск** — курсы фильтруются по категориям, уровню, формату. Аналог: фильтрация курсов по специальности, семестру, преподавателю.
- **Истории студентов** — видео-кейсы "как я стал разработчиком". Аналог: истории выпускников, отзывы.
- **Бесплатные материалы** — "Начните с бесплатного". Аналог: открытые уроки, демо-доступ.

**Паттерны:**
- `сетка карточек` — каталог курсов / специальностей
- `цифры — социальное доказательство` — статистика
- `категории с иконками` — навигация по направлениям
- `фильтры + поиск` — каталог
- `платформа обучения` — ядро LMS
- `сторителлинг / кейсы` — отзывы выпускников

## Выводы для CollegeLMS

На основе анализа референсов, дизайн CollegeLMS должен сочетать:

1. **Академический стиль** (STGAU, VSK) — для официальных разделов: документы, ведомости, отчётность
2. **Современную образовательную платформу** (Netology) — для основного интерфейса: каталог курсов, дашборд, поиск
3. **Фирменные цвета из логотипа** (College Blue + Leaf Green) — как у T-Bank и VTB: один цвет = ДНК
4. **Человечный tone of voice** (T-Bank) — ошибки, пустые состояния, подсказки
5. **Интерактивность** (MTS Bank, T-Bank) — калькуляторы, микро-анимация, персонализация
