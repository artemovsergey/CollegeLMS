# UI/UX Redesign — CollegeLMS

## 1. Дизайн-система

### Цвета
- **Фон**: `#ffffff` (светлая) / `#000000` (тёмная)
- **Текст**: `#000000` / `#ffffff`
- **Акцент (primary)**: первый цвет из выбранной палитры. Используется для ссылок и кнопок.
- **Промежуточные**: остальные цвета палитры (2-4) — для фонов секций, бордеров, ховеров, карточек.
- **База**: всегда белый фон / чёрный текст (светлая) или чёрный фон / белый текст (тёмная).
- Переключение между 5 палитрами через пресеты (data-theme):
  1. Синий: `#568edd` → `#81abe6` → `#abc7ee` → `#d5e3f7`
  2. Индиго: `#24386a` → `#5b6a90` → `#929cb5` → `#c9ceda`
  3. Сапфир: `#4d74b4` → `#7a97c7` → `#a6bada` → `#d3dded`
  4. Сливовый: `#b9b3e5` → `#cbc6ec` → `#dcd9f2` → `#eeecf9`
  5. Зелёный: `#2f8733` → `#63a566` → `#97c399` → `#cbe1cc`

### Типографика
- Основной шрифт: Inter (Google Fonts)
- Размеры: база 16px, h1 — 2rem/32px, h2 — 1.5rem/24px, мелкий текст — 0.875rem/14px
- В будущем — пресеты шрифтов (пользователь выбирает в настройках)

### Логотип + название
- Рядом с SVG-логотипом полное название в две строки:
  ```
  Ставропольский колледж связи
  имени Героя Советского Союза В.А. Петрова
  ```
- Отображается на всех страницах (публичные + аутентифицированные)
- На мобильных — тоже две строки, меньший кегль

## 2. Публичная часть

### Header
- Фиксированный, белый фон, нижняя граница
- Слева: логотип + полное название
- Справа: навигация (горизонтальные ссылки разделов) + кнопка «Войти» / аватарка
- Адаптивность: на мобильных — бургер-меню

### Footer
- Структура как у TPU (из предыдущей реализации):
  - Accordion «Полезная информация» с колонкой ссылок
  - 4 колонки: лого+описание, приёмная комиссия, реквизиты, документы
  - Нижняя плашка: копирайт + ссылки

### PartnersSection
- Существующая структура (иконка + название + описание) — без изменений

### News cards
- Bordered grid как в TPU: изображение, красный тег категории, дата, заголовок
- Сетка с границами между карточками

### Carousel (главная)
- Надпись снизу на изображении — добавить вертикальный градиент (чёрный → прозрачный) снизу вверх, чтобы белый текст читался
- Убрать прозрачный фон у текста, только градиент поверх изображения

### Анимированный лоадер
- При загрузке данных (API, переходы) — SVG логотипа анимируется через stroke-dashoffset
- Контур прорисовывается от начала до конца, затем пульсация
- Цвет — акцентный

## 3. Аутентифицированная часть (LMS)

### Layout
- Слева: drawer-меню (выезжает по кнопке-бургеру)
- Справа: drawer-профиль пользователя (выезжает по клику на аватарку)
- Центр: контент страницы (освобождён от боковых панелей)
- Drawer-меню содержит навигацию по разделам LMS
- Drawer-профиль содержит: ФИО, email, ссылка на профиль, выход

### Страница авторизации
- Centred card (стиль id.cloud.ru)
- Поля: логин, пароль
- Кнопка «Войти» акцентного цвета
- Логотип + название колледжа над формой
- Сообщение об ошибке под формой
- Ссылка «Забыли пароль?»

## 4. Баги

### Импорт WP
- Прогресс-бар показывает 6/100 затем скачет на `6 из 4314%`
- Причина: сервер PHP возвращает `total` в процентах (число от 0-100), а клиент интерпретирует это как количество
- Фикс: сервер должен возвращать `processed` и `total` как числа, клиент считает процент

### Карусель
- Белый текст не читается на светлых участках изображения
- Фикс: добавить `background: linear-gradient(to top, rgba(0,0,0,0.7) 0%, transparent 60%)` на нижнюю часть слайда
